<Alt + F1>：打开主菜单

<Alt + F2运行>：（重启x窗口：r 重启：reboot 关机：hAlt）

<Ctrl + Alt + d>：显示桌面


Print Screen截取全屏

Alt + Print Screen截取窗口`

<Ctrl+Alt+上下箭头>：切换工作区(Fedora)

<Alt+F10>：调整窗口的默认大小(Fedora)
